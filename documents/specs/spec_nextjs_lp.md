# LP仕様書（Next.js）

## 概要

Obsidianで仕様を書いて、Claude CodeでNext.jsのLPを一気に生成するデモ用プロジェクト。

---

## 技術スタック

| 項目 | 技術 |
|------|------|
| フレームワーク | Next.js (App Router) |
| スタイリング | Tailwind CSS |
| テスト | Vitest + Testing Library |

---

## ページ構成

1. **ヒーローセクション** - キャッチコピー + CTAボタン
2. **特徴紹介** - 3カラムレイアウト
3. **料金プラン** - 3プラン構成
4. **FAQ** - アコーディオン形式
5. **CTA + フッター** - 最終アクション誘導

---

## デザイン要件

- モダンでクリーンなデザイン
- レスポンシブ対応（モバイル、タブレット、PC）
- Tailwind CSS使用

---

## コンポーネント構成

```
app/
├── page.tsx           # メインLP
├── layout.tsx         # レイアウト
└── components/
    ├── Hero.tsx       # ヒーローセクション
    ├── Features.tsx   # 特徴紹介
    ├── Pricing.tsx    # 料金プラン
    ├── FAQ.tsx        # FAQ
    └── Footer.tsx     # フッター
```

## テスト要件

### Pricingコンポーネント

- 3つのプランが表示されること
- 各プランに価格が表示されること
- CTAボタンがクリックできること

### テストファイル配置

```
__tests__/
└── Pricing.test.tsx
```

## ワークフロー

1. Obsidianで仕様作成
2. Claude Codeで生成
3. テストコード生成
4. テスト実行
5. プレビュー確認
6. GitHubにcommit
